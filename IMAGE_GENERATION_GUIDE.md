# Image Generation Guide for Portfolio Background Images

## Instructions

Use AI image generators (like DALL-E, Midjourney, Stable Diffusion, or free tools like Recraft.ai, Canva AI, or Bing Image Creator) to generate images based on the prompts below.

After generating the images:
1. Save them with the names: `work-scada.png`, `work-tailing-dams.png`, `work-gnss.png`
2. Place them in the `public/` folder
3. The code has already been updated to use these new images

## Image Generation Prompts

### 1. SCADA Systems Article
**File name:** `work-scada.png`
**Prompt:**
```
Industrial SCADA control room with multiple monitor screens displaying real-time data visualizations, control panels with buttons and switches, modern automation interface, professional industrial setting, clean and technical aesthetic, dark theme with colorful data displays
```

### 2. Tailing Dams Article  
**File name:** `work-tailing-dams.png`
**Prompt:**
```
Aerial view of a large tailing dam at a mining site, industrial infrastructure, earthworks and water management, mining operation landscape, professional engineering photography, dramatic landscape, industrial scale
```

### 3. GNSS Technology Article
**File name:** `work-gnss.png`
**Prompt:**
```
GPS satellite in space orbiting Earth, satellite technology, GNSS monitoring equipment on ground, global positioning system concept, modern technology, professional scientific illustration, clean technical aesthetic
```

3. **Recraft.ai** - https://recraft.ai/
4. **Craiyon** - https://www.craiyon.com/
5. **Stable Diffusion Online** - https://stablediffusionweb.com/

## Image Specifications

- **Format:** PNG or JPG
- **Recommended Size:** 1200x1200px or larger (square format works best for cards)
- **Aspect Ratio:** 1:1 (square) recommended
- **File Size:** Optimize to under 500KB for web performance

## After Adding Images

Once you've added the three new images to the `public/` folder, the code will automatically use them. Refresh your development server to see the changes.
